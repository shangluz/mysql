#一个功能完整的运输企业运营管理系统，基于 Flask + MySQL 技术栈开发，实现运输业务全流程数字化管理。

##功能特性

###用户管理

多角色权限控制：管理员/司机两级权限体系

用户注册登录：安全的认证机制

个人信息管理：用户资料维护

###运输业务管理

全流程任务管理：申请 → 安排 → 执行 → 完成

智能调度系统：自动匹配车辆和司机

实时状态跟踪：任务进度可视化

数据完整记录：计划与实际数据对比

###车辆管理

车辆信息维护：车牌、类型、状态管理

状态实时更新：空闲/任务中/维修状态

历史任务查询：车辆使用记录追踪

###司机管理

司机档案管理：基本信息、联系方式

任务分配记录：司机任务历史查询

绩效数据统计：运输里程、任务次数

###事故管理

事故上报系统：快速记录事故信息

处理流程跟踪：原因分析、处理方式、费用记录

关联责任管理：车辆、司机关联记录

###智能统计分析

运营数据看板：任务量、里程、油耗统计

多维度分析：按时间、车辆、司机、状态分组

数据导出功能：一键导出CSV报表

自定义筛选：灵活的条件查询

##技术栈

###后端技术

框架：Flask 2.3.3

ORM：SQLAlchemy 2.0 + Flask-SQLAlchemy

认证：Flask-Login

数据库：MySQL 8.0+

驱动：PyMySQL

###前端技术

UI框架：Bootstrap 5.3

模板引擎：Jinja2

响应式设计：移动端优先

交互效果：原生JavaScript

###开发工具

开发环境：Python 3.8+

数据库工具：MySQL Workbench

版本控制：Git

代码规范：PEP 8

###环境要求

Python 3.8 或更高版本

MySQL 8.0 或更高版本
